# Freemarker Template Language Support

_This is a vscode extension for [Freemarker Template Language](https://freemarker.apache.org/docs/index.html), via the [LSP(Language Server Protocol)](https://microsoft.github.io/language-server-protocol/)._

> This extension is not fully-featured and can be buggy since it is still in development.

## Features

- Auto-completion
- Hover info
- Goto Definition
- Diagnostics
- Syntax highlighting
